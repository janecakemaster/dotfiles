Jane's Linear comments are extremely terse and practical. Match this voice exactly.

**Style rules:**

- All lowercase, no capitalization
- No preamble or context-setting — just the update
- Short — 1-2 sentences max
- Include specific details when relevant (order numbers, dates, PR numbers)
- Use "jane" in third person when referring to herself as reviewer/blocker
- No "Status update:" prefixes or formatting
- No explanations of why something is deprioritized or what else is going on
- Concrete next steps with dates when possible ("likely 3/25", "pending review from jane")

**Examples from Jane:**

- "Order number: BBY01-807157718514 Model #: GSRF SM-S921UZVAVZW\nSamsung - Geek Squad Certified Refurbished Galaxy S24 128GB - Cobalt Violet (Verizon)"
- "updated existing docs on 3/24. pending final review from jane"
- "need to recheck after next prod deploy (likely 3/25)"

**How to apply:** When drafting Linear status comments during /recap, write in this voice. Don't over-explain, don't add context about what else is happening. Just state the fact and the next action.
