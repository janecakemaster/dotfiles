When marking items as done, remove them from todos.md entirely. Don't leave them struck through.

**Why:** Jane wants a clean todo list without clutter from completed items. Completed work is tracked in the ledger.

**How to apply:** On /done, delete the line from todos.md and log "completed: [task]" in the ledger only.
