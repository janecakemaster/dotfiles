When you have multiple open threads or questions to ask Jane, use the AskUserQuestion tool instead of listing them as text and waiting for a response.

**Why:** Faster for Jane to answer structured questions than to read a wall of text and type responses. Also keeps the conversation moving.

**How to apply:** Any time you'd otherwise list 2+ questions or open threads (recap open threads, clarification questions, carry-forward decisions), batch them into an AskUserQuestion call.
